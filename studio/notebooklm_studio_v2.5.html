📋 Como usar

list --json Lista todos os notebooks. Cole o resultado aqui no Claude para ele interpretar.
use <id> Seleciona um notebook pelo ID (os primeiros 6 caracteres bastam). Sempre execute antes de qualquer operação específica.
source list Lista as fontes do notebook selecionado.
ask "pergunta" Faz uma pergunta ao NotebookLM sobre o conteúdo das fontes.
generate report Gera um relatório. Pode demorar alguns minutos. Depois use artifact wait com o ID retornado.
artifact wait <id> Aguarda a conclusão de um relatório ou podcast em geração.
download report ./arquivo.md Baixa o relatório gerado para o Colab. Você pode então enviar o arquivo aqui para o Claude interpretar.
